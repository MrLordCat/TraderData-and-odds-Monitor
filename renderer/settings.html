Эта настройка применяется глобально. Слушатели получают событие и могут менять парсеры и отображение.

Hotkeys

  • Tab — переключение между буками и Stats
  • F1 — включить/выключить Auto (работает когда окно активно)
  • F2 — Auto Resume after Suspend (toggle)
  • F3 — запуск/остановка Excel/Python скрипта
  • Num5 — глобально (даже вне приложения) toggle Auto
Tab/F1/F2/F3 обрабатываются только когда приложение активно. Num5 — глобальный хоткей (если зарегистрирован OS).

Auto Odds

Tolerance: diff% below this considered aligned. Shock: when Excel odds jump ≥ this %, auto suspends. Burst levels: when diff% ≥ threshold -> use pulses before confirm (F22). First matching level used.

Stats Table

Win/Lose highlight updates instantly.

Table Animations

Анимации пока минимальны (фреймворк заложен). Параметры сохраняются для будущих эффектов.

Activity Heat Bar

Activity bar: increases on events, decays over time. Colors blend; opacity scales max intensity.

Dev
